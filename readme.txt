The Innards

Upon access, this website loads 'js/load.js' to perform an AJAX request to 'data/all.json'. The JSON file contain attributes referencing all content in 'resources/images'. As the content is received, the website displays entire repertoire of work in the body. The footer is a jQuery menu.

The aesthetics is motivated by the mobile trend of app drawers. #EE2A7B is used to focus. #2278BE is used to navigate.
